The purpose of this project is to create a high fidelity prototype for HCI thats primarily aimed at addressing Stress and Burnout. 
Named Pace, as it relates as a to do list which was inspired from thr app todoist but more focused on recognizing user stress levels
as well as reminding of them in these circumstances. This is a basic web format that may or may not include the features from the submitted
research paper during the pre-finals, these include biometrics and wearable device integration which is far more complex.
